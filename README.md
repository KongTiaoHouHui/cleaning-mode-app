# 💻 电脑清洁模式 (Cleaning Mode)

一键锁定屏幕和键盘，让你安心清理设备！

## ✨ 功能特点

- **全屏黑屏锁定**：防止在擦拭屏幕或清理键盘时产生任何误触。
- **高级质感 UI**：带有毛玻璃效果和动态光影的设置界面。
- **动态环形倒计时**：长按退出时，伴有流畅的 HUD 环形进度条和完美同步的倒计时。
- **自定义防误触时间**：可自由设置长按退出的秒数（默认 3 秒）。
- **跨平台**：支持 macOS 和 Windows。

## 📥 下载安装

请前往网页右侧的 **Releases** 区域，下载最新版本：
- **Mac 用户**：下载 `.dmg` 安装包
- **Windows 用户**：下载 `.exe` 安装包

## ⚠️ 常见安装问题 (必看)

由于本软件是免费开源项目，未缴纳苹果/微软官方的高昂“开发者认证费”，因此系统可能会在初次打开时出现拦截提示。请放心，本代码 100% 安全开源，可按以下方法一键解封：

### 🍎 Mac 用户提示“无法验证开发者”或“已损坏”？
**只需两步轻松解决：**
1. **右键打开法**：在应用程序里，按住键盘的 `Control` 键不放，然后**右键点击**软件图标，选择 **“打开”**，并在弹窗中再次点击“打开”即可永久放行。
2. **系统设置法**：如果上述不行，请前往 Mac 的“系统设置” -> “隐私与安全性”，向下滑动找到被拦截的提示，点击 **“仍要打开”**。

### 🪟 Windows 用户弹出蓝色安全提示？
**解决方法**：点击提示窗口左上角的 **“更多信息”**，然后点击右下角出现的 **“仍要运行”** 即可。
